# External Packages Checkpoint

## 📦 Packages Used:
1. **AOS (Animate On Scroll)** - Scroll animations via CDN
2. **Lodash** - JavaScript utility library installed via npm
3. **@emailjs/browser** - Email sending via CDN

## 🚀 Features:
- Scroll-triggered animations
- Dark/Light mode toggle
- Mobile responsive hamburger menu
- Form validation with success messages
- Animated counter statistics
- npm package management
